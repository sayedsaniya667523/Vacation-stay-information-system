# Accommodation Listing & Review Platform

An Airbnb-like web application that allows users to explore tourist accommodations, view prices and details, and read/write reviews. This project focuses on listing, discovery, and reviews.

======

# Features

Browse accommodation listings with price, location, and amenities.

View detailed information for each listing.

User authentication (Register / Login / Logout)

Add, edit, and delete reviews.

Flash messages for user feedback (success/error)

Server-side validation using Joi.

Clean and dynamic UI using EJS templates.
